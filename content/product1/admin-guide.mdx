## Daily Checklist

| Task | Description | Owner |
| --- | --- | --- |
| Review health dashboard | Confirm services report "Healthy" across regions. | On-call admin |
| Verify backups | Ensure overnight snapshots completed successfully. | Storage lead |
| Rotate API tokens | Rotate any expiring credentials within 7 days. | Security team |

## Escalation Steps

1. Capture relevant logs from the orchestrator nodes.
2. Create an incident ticket in the "Critical" queue.
3. Notify stakeholders via the #product1-ops Slack channel.
